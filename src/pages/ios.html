Swift 3.0

Swift is a powerful and intuitive programming language for iOS apps that builds on the best of C and Objective-C, without the constraints of C compatibility. Swift adopts safe programming patterns and adds modern features to make programming easier, more flexible, and more fun. A Swift 3.0 roadmap was announced on the Swift blog on December 3, 2015. Swift 3.0 is the first major release developed in the open at Swift.org, with source code, a bug tracker, mailing lists, and regular development builds available for everyone.

Know more about Swift at: https://developer.apple.com/swift/

GIT

GIT was initially designed and developed by Linus Torvalds for Linux kernel development. GIT is a free software that is designed to handle distributed revision control and source code management system. Easy to learn, it manages anything from small to large projects with speed and efficiency. Every Git clone is a full-fledged repository with complete history. It possess full revision tracking capabilities that are independent of network access or a central server.

Download link: https://git-scm.com/downloads

FASTLANE

Fastlane simplifies and automates app deployment and release process within a single streamlined workflow thereby ensuring Continuous Delivery (CD). App deployment occurs in a single push of a button using the Fastlane tools by creating a custom list of sequential build tasks called lanes. This improves the development time by accelerating software quality and reaction time with more frequent and smaller releases that can be deployed by anyone in the team. It also automates tedious tasks like generating screenshots, dealing with provisioning profiles, and releasing the application to the App Store.

Download link: https://github.com/fastlane/fastlane

TESTFLIGHT

TestFlight Beta Testing is an Apple product that makes it easy to invite users to test iOS, watchOS and tvOS apps before it is released into the App Store. It is exclusively offered to developers within the iOS Developer Program. The Developers who subscribe for the service can distribute their apps to internal or external beta testers. These testers can in turn send feedback on the apps to the developers, thereby improving the testing time and reaction time. Besides Tester Feedback, the TestFlight SDK also allows the developers to receive remote logs and crash reports of the apps.

Download link: https://developer.apple.com/testflight/

BAAS

Backend as a Service, or BaaS (sometimes referred to as mBaaS) is a cloud computing category that make it easier for developers to setup, use and operate a cloud backend for their mobile, tablet and web apps.It provides web and mobile app developers with a way to link their applications to the cloud storage. They also add features such as user management, push notifications, and integration with social networking services for enhancement. These services are provided through customized Software Development Kits (SDKs) and Application Programming Interfaces (APIs).

SpriteKit

SpriteKit provides a graphics rendering and animation infrastructure that can be used to animate arbitrary textured images, or sprites. SpriteKit is optimized so that the positions of sprites can be changed arbitrarily in each frame of animation. It also provides other functionality that is useful for games, including basic sound playback support and physics simulation. The combination of framework and tools makes SpriteKit a good choice for games and other apps that require similar kinds of animation.

Download link: https://developer.apple.com/SpriteKit_PG/Introduction/Introduction.html
